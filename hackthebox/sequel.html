HackTheBox: Sequel

Summary

Sequel is a beginner-friendly Linux machine that focuses on Vulnerability Management, Databases, MySQL, SQL, Reconnaissance, and Weak Credentials.

TASK 1: During our scan, which port do we find serving MySQL?

3306

nmap -sV -F {ip address}

Task 2: What community-developed MySQL version is the target running?

MariaDB

nmap -sV -sC -v {ip address}

Task 3: When using the MySQL command line client, what switch do we need to use in order to specify a login username?

-u

Can be obtained through a simple Google search.

Task 4: Which username allows us to log into this MariaDB instance without providing a password?

root

Can be obtained through a simple Google search.

Task 5: In SQL, what symbol can we use to specify within the query that we want to display everything inside a table?

*

Can be obtained through a simple Google search.

Task 6: In SQL, what symbol do we need to end each query with?

;

Can be obtained through a simple Google search.

Task 7: There are three databases in this MySQL instance that are common across all MySQL instances. What is the name of the fourth that's unique to this host?

htb

Lessons Learned

  • Enumerating open ports using nmap helped identify a MySQL database service running on the target machine.
  • Connecting to the MySQL service without a password exposed the database schema and enabled extraction of user credentials.
  • Learned how to extract and crack MySQL password hashes using tools like hashcat or John the Ripper.
  • Pivoted from database access to system shell using known credentials through SSH.
  • Performed privilege escalation by exploiting SUID binaries or weak file permissions.
  • Reinforced the importance of securing database services and removing anonymous or empty-password accounts.
