HackTheBox: Nibbles

📝 Summary

Nibbles is a beginner-friendly Linux machine that focuses on enumeration, exploiting a vulnerable web app, and gaining root through a local privilege escalation exploit.

🔍 Enumeration

nmap -sC -sV -oA nibbles 10.10.10.75

We find an HTTP service on port 80 and a custom web app on /nibbleblog.

🔓 Exploitation

We gain access to the admin panel using default credentials:

admin:nibbles

We then upload a PHP reverse shell via the plugin feature and get a reverse shell:

nc -lvnp 4444

📈 Privilege Escalation

Using sudo -l we discover that we can run a script as root:

sudo /home/nibbler/personal/stuff/monitor.sh

We modify the script to get a root shell.

🏁 Root Flag

cat /root/root.txt

🎯 Lessons Learned

  • Importance of web app enumeration
  • Using default creds to gain initial access
  • Identifying privilege escalation vectors through sudo permissions
