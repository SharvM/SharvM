Lessons Learned

  • Performed enumeration and found only port 80 (Apache httpd)
  • Used gobuster for directory brute‑forcing to explore webapp structure
  • Identified an SQL injection vulnerability in the login form/search functionality
  • Exploited SQLi to bypass authentication (e.g., using payloads like admin'#)
  • Extracted sensitive data such as usernames and hashes via database queries
  • Learned the importance of sanitizing user input and using parameterized queries to prevent SQL injection
  • Gained insight into how web app misconfigurations can be chained to escalate initial access
